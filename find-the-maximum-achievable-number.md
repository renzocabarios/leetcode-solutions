## find-the-maximum-achievable-number

## REFERENCE

- https://leetcode.com/problems/find-the-maximum-achievable-number/

## SOLUTION

``` javascript
function theMaximumAchievableX(num: number, t: number): number {
    return num + (2 * t);
};
```
